## 🎯 Como Abrir o Portfólio

### Método 1: Duplo Clique
1. Navegue até a pasta do projeto
2. Clique duas vezes no arquivo `index.html`
3. O portfólio abrirá no seu navegador padrão

### Método 2: Pelo Navegador
1. Abra seu navegador (Chrome, Firefox, Edge, etc.)
2. Pressione `Ctrl + O` (ou `Cmd + O` no Mac)
3. Navegue até a pasta do projeto
4. Selecione o arquivo `index.html`
5. Clique em "Abrir"

### Método 3: Arrastar e Soltar
1. Abra seu navegador
2. Arraste o arquivo `index.html` para a janela do navegador
3. Solte o arquivo

## 🔧 Personalização

### Para Personalizar o Portfólio:

1. **Alterar Informações Pessoais:**
   - Edite o arquivo `index.html`
   - Modifique as seções correspondentes

2. **Modificar Traduções:**
   - Edite o objeto `translations` no arquivo `script.js`
   - Adicione novas chaves conforme necessário

3. **Ajustar Cores e Estilos:**
   - Modifique as variáveis CSS no arquivo `style.css`
   - Altere as cores, fontes e espaçamentos

4. **Adicionar Novas Seções:**
   - Adicione o HTML da nova seção
   - Inclua os estilos CSS correspondentes
   - Adicione as traduções no JavaScript
   - Atualize o menu de navegação
